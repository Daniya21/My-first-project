# Informatics Calculator

A feature-rich calculator application developed in Python that performs mathematical calculations, statistical operations, number system conversions, and user authentication. This project was created to strengthen programming fundamentals, problem-solving skills, and practical implementation of Python concepts.

## Features

### User Authentication

* User registration system
* Secure login functionality
* Stores user credentials in a text file
* Validates user access before allowing calculator operations

### Arithmetic Operations

* Addition
* Subtraction
* Multiplication
* Division with zero-division handling

### Mathematical Functions

* Square Root Calculation
* Mode Calculation
* HCF (Highest Common Factor)
* LCM (Lowest Common Multiple)
* Multiples Less Than 50

### Statistical Operations

* Mean Calculation
* Median Calculation

### Number System Conversion

* Decimal to Binary
* Decimal to Octal
* Decimal to Hexadecimal
* Binary to Decimal
* Octal to Decimal
* Hexadecimal to Decimal

### Error Handling

* Input validation
* Invalid number detection
* Division by zero protection
* Invalid menu choice handling

## Technologies Used

* Collections Module
* File Handling
* Functions
* Loops and Conditional Statements
